## Overview

Lumina Bible App offers an immersive reading and study environment. It incorporates an extensive audio library, enabling users to listen to their study material entirely offline.

## Key Features

- Offline audio playback
- Study notes and bookmarking system
- Lightweight and fast on low-end devices

## Technologies Used

- React Native
- Firebase
- Audio API

## My Role

Spearheaded the integration of the Audio API with offline storage, ensuring uninterrupted access to study resources even without internet connectivity.

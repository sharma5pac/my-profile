## Overview

The NurQuran App provides a spiritually enriching and intuitive mobile experience. Designed with a stunning dark theme, it seamlessly combines essential daily Islamic features with a dedicated, elegant interface for reading the Holy Quran.

## Key Features

- **Daily Blessings:** A home dashboard featuring the next prayer times, Athan reminders, and a carefully selected Dua of the Day.
- **Holy Quran Reading:** Read and listen to the words of Allah with a beautifully organized Surah directory including Al-Fatihah, Al-Baqarah, and more.
- **Integrated Tooling:** Seamlessly switch between Home, Quran, NurAI, and Hifz tracking.

## Technologies Used

- React Native
- Expo
- REST APIs & Audio Services
